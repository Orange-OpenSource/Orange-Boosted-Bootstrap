# Component versioning

OUDS components are first designed on Figma, then developed for each platform. There can be a delay between a component's availability on Figma and on OUDS Web.

Each component in the documentation lists its "design version" so developers can check parity with the Figma design.

Component design versions are fixed for a specific OUDS Web release. Update the library to benefit from updated components.
