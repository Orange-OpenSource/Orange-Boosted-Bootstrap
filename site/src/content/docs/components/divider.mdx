## Horizontal rules

Horizontal rules should use the `<hr />` tag, when the separator has semantic meaning (i.e., when it represents a topic change or a significant transition in the content). If the divider is purely decorative, prefer using a border CSS property on an HTML element or our [border utilities]([[docsref:/utilities/border]]), or even ensure to hide it from assistive technologies (e.g., `aria-hidden="true"`).

<Example buttonLabel="horizontal rule" code={`<hr />
<div class="border-top border-default my-medium"></div>`} />

## Vertical rules

Vertical rules are inspired by the `<hr />` element. The `vr` class allows you to create vertical dividers in common layouts. Theyâ€™re styled just like `<hr />` elements. They have `min-height` of `1em`.

<Example buttonLabel="vertical rule" code={`<div class="vr"></div>`} />

Vertical rules scale their height in flex layouts:

<Example buttonLabel="vertical rule with flex" class="d-flex" code={`<div class="d-flex" style="height: 50px;">
  <div class="vr"></div>
</div>
<div class="d-flex" style="height: 80px;">
  <div class="vr"></div>
</div>
<div class="d-flex" style="height: 100px;">
  <div class="vr"></div>
</div>`} />

## Color

Change the divider color using the [border color utilities]([[docsref:/utilities/border#color]]).

<Example buttonLabel="colored horizontal rules" class="d-flex flex-column gap-2xsmall" code={[...allTokens.filter(token => token.name.match(/\$ouds-color-border-/) && token.name.match(/-light/) && token.compiledValue && !token.name.match(/-focus/)).map(token => {
    if (token.name.match(/-on-/)) {
      return `<div class="bg-${token.name.split('-').slice(4, -1).join('-')} py-2xsmall"><hr class="border-${token.name.split('-').slice(3, -1).join('-')}"></div>`
    }
    return `<hr class="border-${token.name.split('-').slice(3, -1).join('-')}" />`
  }),
  `<div class="bg-always-white py-2xsmall"><hr class="border-always-black"></div>
  <div class="bg-always-black py-2xsmall"><hr class="border-always-white"></div>
  <div class="bg-always-black py-2xsmall"><hr class="border-always-on-black"></div>
  <div class="bg-always-white py-2xsmall"><hr class="border-always-on-white"></div>`]} />

<Example buttonLabel="colored vertical rules" class="d-flex flex-column gap-2xsmall" code={[...allTokens.filter(token => token.name.match(/\$ouds-color-border-/) && token.name.match(/-light/) && token.compiledValue && !token.name.match(/-focus/)).map(token => {
    if (token.name.match(/-on-/)) {
      return `<div class="bg-${token.name.split('-').slice(4, -1).join('-')} py-2xsmall"><div class="vr border-${token.name.split('-').slice(3, -1).join('-')}"></div></div>`
    }
    return `<div class="vr border-${token.name.split('-').slice(3, -1).join('-')}"></div>`
  }),
  `<div class="bg-always-white py-2xsmall"><div class="vr border-always-black"></div></div>
  <div class="bg-always-black py-2xsmall"><div class="vr border-always-white"></div></div>
  <div class="bg-always-black py-2xsmall"><div class="vr border-always-on-black"></div></div>
  <div class="bg-always-white py-2xsmall"><div class="vr border-always-on-white"></div></div>`]} />

<BootstrapCompatibility>
The `<hr />` and `.vr` color can be set using [colors utilities]([[docsref:/utilities/color]])
<Example buttonLabel="colored rules (Bootstrap compatibility)" code={`<hr class="text-muted">
<div class="vr text-brand-primary"></div>`} />
</BootstrapCompatibility>

## Size

Change the divider thickness using the [border width utilities]([[docsref:/utilities/border#width]]).

<Example buttonLabel="horizontal rule sizes" code={`<div>
    <hr class="border-thin">
    <hr class="border-medium">
    <hr class="border-thick">
    <hr class="border-thicker">
  </div>`} />

<Example buttonLabel="vertical rule sizes" code={`<div class="d-flex" style="height: 50px;">
    <div class="vr border-thin"></div>
    <div class="vr border-medium"></div>
    <div class="vr border-thick"></div>
    <div class="vr border-thicker"></div>
  </div>`} />
