## Base class

OUDS Web has a base `.badge` class that sets up basic styles.

{{< example >}}
<span class="badge"><span class="visually-hidden">Badge alt text</span></span>
{{< /example >}}

## Standard variants 

<!-- TODO: set the right url for design documentation -->
OUDS Web [color and background helpers](/docs/{{< param docs_version >}}//helpers/color-background) can be applied to display badge variants, each serving its own semantic purpose. Please follow the [Badge design specifications](https://unified-design-system.orange.com/) to choose the right badge for the right action.

{{< example class="d-flex gap-3" >}}
<span class="badge"><span class="visually-hidden">Badge alt text</span></span>

<span class="badge badge-disabled"><span class="visually-hidden">Badge alt text</span></span>
{{< /example >}}

## With count

Badges can be used to display numerical value using the `.badge-count` class.

{{< example class="d-flex align-items-center gap-3" >}}
<span class="badge badge-count">1</span>
<span class="badge badge-lg badge-count">1</span>
<span class="badge badge-count text-bg-status-positive-emphasized">23</span>
<span class="badge badge-count badge-lg text-bg-status-positive-emphasized">23</span>
<span class="badge badge-count badge-disabled">+99</span>
<span class="badge badge-count badge-lg badge-disabled">+99</span>
{{< /example >}}

### Positioned

Use utilities to modify a `.badge` and position it in the corner of a button with an icon.

{{< example >}}
<button type="button" class="btn btn-icon btn-default position-relative">
  <svg aria-hidden="true">
    <use xlink:href="/docs/{{< param docs_version >}}/assets/img/ouds-web-sprite.svg#trash"/>
  </svg>
  <span class="visually-hidden">Trash</span>
  <span class="position-absolute top-0 start-100 translate-middle badge text-bg-status-negative-emphasized">
    <span class="visually-hidden">Trash contains items</span>
  </span>
</button>

<button type="button" class="btn btn-default position-relative ms-md">
  Inbox
