export const opacityBackgroundStyle = "background-position: 0px 0px, 10px 10px; background-size: 20px 20px; background-image: linear-gradient(45deg, #000 25%, transparent 25%, transparent 75%, #000 75%), linear-gradient(45deg, #000 25%, transparent 25%, transparent 75%, #000 75%)";

## Opacity

The `opacity` property sets the opacity level for an element. The opacity level describes the semantic transparency level, where `opaque` is not transparent at all, `emphasis` to `weaker` are various levels of transparency, and `transparent` is completely transparent.

### Common

Set the opacity of an element using `.opacity-{value}` utilities.

<Example showMarkup={false} class="sm:d-flex" code={`<div class="opacity-opaque p-large m-small fw-bold" style="${opacityBackgroundStyle}"></div>
  <div class="opacity-strong p-large m-small fw-bold" style="${opacityBackgroundStyle}"></div>
  <div class="opacity-medium p-large m-small fw-bold" style="${opacityBackgroundStyle}"></div>
  <div class="opacity-weak p-large m-small fw-bold" style="${opacityBackgroundStyle}"></div>
  <div class="opacity-weaker p-large m-small fw-bold" style="${opacityBackgroundStyle}"></div>
  <div class="opacity-weakest p-large m-small fw-bold" style="${opacityBackgroundStyle}"></div>
  <div class="opacity-invisible p-large m-small fw-bold" style="${opacityBackgroundStyle}"></div>`} />

```html
<div class="opacity-opaque">...</div>
<div class="opacity-strong">...</div>
<div class="opacity-medium">...</div>
<div class="opacity-weak">...</div>
<div class="opacity-weaker">...</div>
<div class="opacity-weakest">...</div>
<div class="opacity-invisible">...</div>
```

The `opacity` property sets the opacity level for an element. The opacity level describes the transparency level, where `1` is not transparent at all, `.5` is 50% visible, and `0` is completely transparent.

Set the `opacity` of an element using `.opacity-{value}` utilities.
