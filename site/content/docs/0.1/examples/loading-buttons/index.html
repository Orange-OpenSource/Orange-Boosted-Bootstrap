Loading buttons

Find here some examples of how to implement a live loading button, for an indeterminate or determinate time. These examples are not part of the OUDS Web library and must be adapted to your project. You can find the code below.

Loading button in an indeterminate time

For this example, the time is set to 5 seconds by Javascript, but in a real project, you just need a callback when the action is finished. An update to the status is done every second to let the user know the download is still ongoing. Each time, a dot is added to the message to make sure it will change and then be vocalised by the screen readers.

Loading buttons in a determinate time

The time must be set to through the CSS variable --bs-button-loading-time directly on the button. In this example, it is set to 4 seconds. An update to the status is done every second, giving the percentage of time that has elapsed.
